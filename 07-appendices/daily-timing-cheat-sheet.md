# Daily Timing Cheat Sheet

[Back to Appendices](README.md) | [Back to Manual](../README.md)

This is the compressed version of the daily rhythm used across the 28-day cycle. Use it when you remember the structure but do not want to reopen a full day page.

## Default Daily Rhythm

| Time | Action | Default Components | Notes |
| :--- | :--- | :--- | :--- |
| **06:30** | Fasted cardio | 20-40 minutes, low intensity | The goal is signaling, not output. |
| **07:15** | Morning hydration | Ajwain water or jeera water | Choice depends on the day plan. |
| **12:00** | Main meal / break fast | Protein + broccoli + chilled starch + core spice stack | Primary supplement window. |
| **14:30** | Salad layer | Greens + ferment + seeds + microgreens + cilantro | Rotators vary by day. |
| **16:30** | Resistance training | Standard training slot | Use the day page if the day has special emphasis. |
| **17:30** | Post-workout protein | Whey isolate + creatine | Flavored whey stays here only. |
| **17:50** | Recovery bowl | Greek yogurt + casein + blueberries | Cacao is optional and spaced deliberately. |
| **21:00** | Night infusion | Tulsi or saffron | Depends on the day plan. |

## Default Anchors

- **Main meal:** Turmeric + black pepper, broccoli, and a defined protein source.
- **Salad:** Microgreens and cilantro are daily anchors.
- **Recovery:** Keep the whey window and bowl window separate.

## Common Morning Hydration Options

| Option | Common Use |
| :--- | :--- |
| **Ajwain water** | Digestive activation or antimicrobial-leaning days |
| **Jeera water** | Baseline digestive support days |

## Common Night Infusions

| Option | Common Use |
| :--- | :--- |
| **Tulsi infusion** | Lower-stimulation recovery nights |
| **Saffron infusion** | Parasympathetic emphasis and end-of-day downshift |

## When to Reopen a Full Day Page

- A day includes explicit exclusions like "no cacao" or "no pomegranate."
- A day calls for a special rotator like clove, nigella, or rosemary.
- A day changes the vascular or antimicrobial emphasis materially.
