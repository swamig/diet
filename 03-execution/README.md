# Daily Execution

[Back to Manual](../README.md)

This is the operating layer of the manual. Each week guide breaks the 28-day cycle into a focused phase, and each day file contains the exact timing, meals, and rationale for that day.

## Before Day 1

- Read the [Foundations Guide](../01-foundations/README.md).
- Set up the kitchen using [Practical Logistics](../05-practical/README.md).
- Build a shopping and prep baseline with [Shopping List](../05-practical/shopping-list-master.md) and [Sunday Prep](../05-practical/sunday-prep.md).
- Capture a starting snapshot in [Baseline Checklist](../06-monitoring/baseline-checklist.md).
- Decide how you will capture adherence in [Logs](../logs/README.md).

## How to Use a Daily Page

1. Read the objective first so the day has a clear emphasis.
2. Pre-stage any special ingredients, infusions, or exclusions before the feeding window.
3. Follow the timing structure, but log real-world deviations instead of hiding them.
4. Use the clinical rationale as the reason for the day, not just as notes at the bottom.
