# Weekly Prep Checklist

[Back to Practical Logistics](README.md) | [Back to Manual](../README.md)

Use this as the operator checklist that sits beside [Sunday Prep](sunday-prep.md). It is intentionally plain and repetitive so execution does not depend on memory.

## Before Shopping

- [ ] Review the current [Week Overview](../03-execution/README.md).
- [ ] Check the fridge, freezer, and spice inventory.
- [ ] Note substitutions before shopping, not during cooking.

## Shopping

- [ ] Refill greens, herbs, crucifers, and ferments.
- [ ] Refill primary protein anchors.
- [ ] Refill recovery bowl ingredients.
- [ ] Refill spice and infusion ingredients.

## Sunday Prep

- [ ] Chop and rest the crucifer batch.
- [ ] Cook / chill legumes and grains.
- [ ] Prep protein anchors.
- [ ] Wash and dry greens.
- [ ] Portion ferments, seeds, and recovery ingredients.
- [ ] Set up spice jars if using the jar workflow.
- [ ] Whisk and store **Dressing Master Batch** (Olive Oil/Balsamic/Puree/Dijon).

## Storage Reset

- [ ] Place ready-to-eat items on the correct shelves.
- [ ] Replace any wet paper towels.
- [ ] Move shorter-life items to the front.
- [ ] Label anything that will be easy to forget midweek.

## Midweek Reset

- [ ] Refresh greens and towels.
- [ ] Re-chop broccoli if needed.
- [ ] Check tofu water and ferment levels.
- [ ] Refill any missing recovery bowl pieces.

## Weekly Review

- [ ] Update the [Weekly Review Template](../06-monitoring/weekly-review.md).
- [ ] Note repeated substitutions or skipped steps.
- [ ] Decide what needs simplifying before the next week begins.

## Evidence Status

- **General physiology:** The storage, prep, and recovery logic on this page is meant to preserve the intended structure of the protocol rather than prove a clinical outcome on its own.
- **Protocol-specific rules:** The exact prep steps, temperatures, container choices, sequencing, and repetition rules on this page are operating rules specific to this manual.

## Conflicts

- Treating the setup and midweek reset as optional usually pushes friction into weekdays.
- Leaving review work to memory instead of writing it down weakens the feedback loop this checklist is meant to support.
