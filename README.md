# High Signaling Performance Manual 🌿 (Clinical + Performance + Full Biochemical Edition)

This repository is a metabolic operations manual organized around one question: how do you make the protocol easy to understand, execute, and review without losing the biochemical logic underneath it?

The content works best when read as a system instead of as isolated notes.

## Evidence Handling

Mechanism-heavy leaf pages now include an `Evidence Status` section. Where a page has had a source pass, it also includes a `Sources` section. Together, those sections separate general physiology framing from the protocol-specific rules used by this manual, especially around exact doses, timing, rotation frequency, and exclusion logic.

Use [Source Style Guide](07-appendices/source-style-guide.md) when adding new externally supported claims.

## Start Here

1. Read the [Foundations Guide](01-foundations/README.md) to understand the non-negotiable rules.
2. Set up the [Practical Logistics Guide](05-practical/README.md) so daily execution is low-friction.
3. Run the [28-Day Execution Guide](03-execution/README.md).
4. Capture adherence and deviations in [Logs](logs/README.md) and review outcomes in [Monitoring](06-monitoring/README.md).

## Core Protocol Principles

1. **Metabolic Flexibility:** Efficiently switching between glucose and fat oxidation. See [Metabolic Flexibility](01-foundations/01-metabolic-flexibility.md).
2. **Inflammatory Signaling Control:** Mandatory daily stacking of turmeric + black pepper. See [Inflammatory Control](01-foundations/02-inflammatory-control.md).
3. **Sulforaphane Optimization:** The "chop and rest" rule for cruciferous vegetables. See [Sulforaphane Activation Protocol](01-foundations/02-inflammatory-control.md#sulforaphane-activation-protocol).
4. **Microbiome Hormesis:** Rotational exposure to antimicrobial spices. See [Gut Microbiome Diversification](01-foundations/03-microbiome-hormesis.md).
5. **Resistant Starch Cycling:** The cook-and-chill method for legumes and grains. See [Resistant Starch Cycling](01-foundations/05-resistant-starch.md).
