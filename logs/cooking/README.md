# Cooking & Prep Logs

Use this folder to log your Sunday batch prepping sessions, recipe adjustments, spice ratios, and ingredient sourcing notes.

## Things to Track:
- How long prep took.
- Where you sourced high-quality ingredients (e.g., specific farm for microgreens).
- Spice combination tweaks.
- Fermentation times (for things like sauerkraut or if you make your own).
